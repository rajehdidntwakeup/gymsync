# GymSync Mobile (Expo)

Workout tracking app built with Expo and React Native.

## Setup

```bash
cd mobile-expo
npm install
```

## Running

```bash
# Start Expo
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios
```

## API Configuration

Update `app.json` extra.apiUrl:
- Android Emulator: `http://10.0.2.2:8080`
- iOS Simulator: `http://localhost:8080`
- Physical device: Your machine's local IP

## Features

- Workout logging
- Gym finder (maps)
- Real-time chat
- Progress tracking
- User profiles
